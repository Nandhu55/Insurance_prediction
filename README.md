# 💰 Insurance Cost Prediction

A **Machine Learning web application** that predicts insurance premium costs based on customer details such as age, income, policy term, and sum assured.

This project uses **Python, Scikit-learn, and Streamlit** to build an interactive UI where users can enter information and instantly get an estimated insurance cost.

---

## 🚀 Live Application

🔗 **Streamlit App:**  
https://batch-25-insurance-prediction.streamlit.app/

---

## 📌 Features

- Predict insurance premium using Machine Learning
- Interactive **Streamlit UI**
- Clean and modern dashboard interface
- Fast prediction using trained model
- Easy deployment on Streamlit Cloud

---

## 🧠 Machine Learning Model

The model was trained using historical insurance data to learn patterns between customer attributes and insurance costs.

### Input Features

- Age
- Annual Income (LPA)
- Policy Term (Years)
- Sum Assured (Lakhs)

### Output

- Estimated Insurance Cost
